SideJot Privacy Policy

Last updated: 2026-03-27

Overview

SideJot is a browser extension that lets users create, edit, and sync notes in a browser side panel.

This policy explains what data SideJot processes, why it is processed, and how it is stored.

Data We Process

  • User account information: Google sign-in information required for authentication.
  • User-created note content: Notes, titles, timestamps, and related source URLs.
  • Selected webpage content: Text, page URL, and title only when you explicitly use the context menu to save content to SideJot.
  • Local app settings: UI preferences such as panel layout and local extension settings.

How Data Is Used

  • Authenticate users with Google Sign-In
  • Save and organize notes
  • Sync notes across devices using Firebase / Google Cloud services
  • Import and export note data at the user's request
  • Remember local UI preferences

Data Storage

SideJot stores data locally in your browser using extension storage and IndexedDB. If sync is enabled, note data is also stored in Firebase / Google Cloud services for cross-device synchronization.

Data Sharing

SideJot does not sell user data and does not use note content for advertising.

Data may be processed by infrastructure providers strictly to provide app functionality, such as Google Authentication and Firebase / Google Cloud services.

User Control

  • Sign out at any time
  • Delete notes at any time
  • Export notes
  • Remove the extension to stop future local processing

Permissions

SideJot requests only the permissions needed for its features, including side panel access, local storage, active tab access, context menu access, and identity / authentication access.

Security

Reasonable steps are taken to protect stored data, but no method of storage or transmission is 100% secure.

Children's Privacy

SideJot is not intended for children under 13.

Changes to This Policy

This policy may be updated from time to time. Updates will be reflected by the “Last updated” date.

Contact

For privacy-related questions, contact the developer through the Chrome Web Store listing or the published support channel.
